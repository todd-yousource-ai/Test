## Why

The TRD (§2.1) specifies that `__init__.py` is the "package root, re-exports public API." The documentation set (API.md, README.md) establishes the contract that users can write:

```python
from tasklib import Task, TaskStatus, InMemoryTaskStore
```

Without this wiring, users must reach into subpackages directly, violating the documented API contract. This PR closes the gap between the documented public surface and the actual importable symbols.
