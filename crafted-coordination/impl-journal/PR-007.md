## What

Implement the CLI entry point (`src/tasklib/cli.py`) providing three subcommands — `add`, `list`, and `complete` — backed by `InMemoryTaskStore`. Wire the `console_scripts` entry point in `pyproject.toml` so the CLI is available as `tasklib` after install.

## Why

This closes functional requirements FR-15 through FR-19 from TRD-TASKLIB. The CLI is the final user-facing component in the validation chain, proving that the Crafted Dev Agent pipeline can produce a working end-to-end artifact from documentation through scaffold to executable code.

## How

- **`src/tasklib/cli.py`**: Uses only stdlib `argparse` with subparsers for `add`, `list`, and `complete`. Each subcommand handler receives the parsed `argparse.Namespace` and an `InMemoryTaskStore` instance, and returns an integer
