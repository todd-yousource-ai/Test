## What

Implement `InMemoryTaskStore` in `src/tasklib/storage/memory.py` backed by a plain `dict[str, Task]`. The store provides three operations:

- `add(title: str) -> Task` — creates a new `Task` from the given title, stores it by ID, and returns the stored task.
- `get(task_id: str) -> Task` — retrieves a task by its identifier, raising `KeyError` if not found.
- `list_all() -> list[Task]` — returns a list of all stored tasks in insertion order (Python 3.7+ dict ordering).

Re-exports `InMemoryTaskStore` from `src/tasklib/storage/__init__.py`.
