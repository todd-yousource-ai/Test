## What

Creates the repository-root `pyproject.toml` declaring the `tasklib` package build and metadata configuration.

## Why

This is the foundational packaging configuration required before any Python source files are added. Per PRD-002 and TRD-TASKLIB, the project needs a properly configured build system that declares the package identity, Python version constraint, zero external dependencies, src-layout package discovery, and the CLI entry point. Establishing this file first ensures that subsequent PRs adding source scaffolding and implementation can immediately be installed and tested in editable mode.
