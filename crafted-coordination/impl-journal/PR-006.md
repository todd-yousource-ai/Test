## What

Update `src/tasklib/__init__.py` to re-export `Task`, `TaskStatus`, and `InMemoryTaskStore` as the public API surface. Define `__all__` to restrict wildcard imports to exactly these three symbols.

## Why

FR-20 (TRD-TASKLIB §2.1) specifies that the package root re-exports the public API. Without this wiring, consumers must reach into sub-packages (`tasklib.models.task`, `tasklib.storage.store`) to access core types. This PR closes the last link in the import chain so that the idiomatic usage pattern works:

```python
from tasklib import Task, TaskStatus, InMemoryTaskStore
```

This is a prerequisite for the CLI module and any downstream consumer code.

## How

1. Import `Task` and `TaskStatus` from `tasklib.models.task`.
2. Import `InMemoryTaskStore` from `tasklib.storage.store`.
