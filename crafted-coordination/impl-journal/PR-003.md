## What

Creates the `src/tasklib/` directory tree with three `__init__.py` files:

- `src/tasklib/__init__.py` — package root
- `src/tasklib/models/__init__.py` — models subpackage
- `src/tasklib/storage/__init__.py` — storage subpackage

## Why

This is PR #3 in the tasklib dependency chain, required by PRD-002 (Package Scaffold). Every downstream PR (Task model, Task store, CLI) depends on these three importable package paths existing. Without this scaffold, no Python module can be placed into `tasklib.models` or `tasklib.storage`.

This PR is deliberately separated from `pyproject.toml` configuration (a different PR) and from placeholder modules (`task.py`, `store.py`, `cli.py`) to keep each PR single-purpose and independently mergeable.
