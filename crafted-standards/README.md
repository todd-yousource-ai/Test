# Crafted Engineering Standards

This directory is automatically created by Crafted Dev Agent on every feature branch.

## Contents

| File | Purpose |
|---|---|
| `ENGINEERING_STANDARDS.md` | Full Crafted Engineering Standards (inline copy) |
| `crafted_engineering_standards.zip` | Authoritative standards archive |
| `../docs/BRANCH_CONTEXT.md` | Task description and consensus decision for this branch |

## Why this exists

Every branch that contains AI-generated code must carry the full engineering
standards as a co-located artifact. This ensures:

- Code reviewers always have the standards in context
- AI assistants (Copilot, Claude, etc.) can reference them inline
- The PR audit trail is self-contained and portable
- Standards compliance can be verified without external lookups

## Usage

To verify a file against Crafted standards, reference `ENGINEERING_STANDARDS.md`
in your review prompt or Copilot context.
